## Projects

**Tenere — macOS Focus and Attention App**

A native macOS application that monitors browser and application activity in real time using CGEventTap and NSWorkspace APIs, scores focus quality against user-declared work intentions, and applies a violation detection system that distinguishes genuine interruptions from prolonged distractions. Built with a full CI/CD pipeline using GitHub Actions and Docker.

`Swift` `SwiftUI` `Supabase` `Docker` `GitHub Actions`

---

**Enterprise Social Listening Platform**

A centralised analytics platform that aggregates content from multiple social media REST APIs into a MySQL backend, with NLP pipelines for sentiment analysis, controversy detection, and keyword clustering. Output surfaces through Streamlit dashboards with alerting for engagement spikes.

`Python` `NLP` `MySQL` `Streamlit` `REST APIs`

---

**Busserbot — Autonomous Restaurant Robot**

An autonomous robot designed to navigate dynamic restaurant environments, detect vacated tables using multi-sensor fusion and OpenCV, and execute tray-clearing routines without human input. Navigation uses A* pathfinding with dynamic obstacle handling.

`Python` `WeBots` `OpenCV` `A*`
